# hoby/power

Next.js project with E2E testing.

## Startup
```bash
npm run dev
```

## Testing
Run E2E tests (Playwright):
```bash
npm test
```

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Testing**: Playwright

## Directory Structure
- `src/`: Application source code
  - `app/`: App Router pages and layouts
- `e2e/`: E2E test files
- `public/`: Static assets
